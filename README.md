Pipeline actuel :

Vidéo brute
↓
Annotation utilisateur
↓
Transformation en événement structuré
↓
Extraction features (zone, phase, minute, contexte)
↓
Moteur explicatif heuristique
↓
Sortie structurée (scores + texte + recommandations)


Le POC démontre qu’à partir de métadonnées structurées, une IA explicative peut générer une analyse causale cohérente.
